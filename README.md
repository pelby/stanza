# Stanza — release binaries

This repository hosts **release binaries** for [Stanza](https://stanza.pub), a lightweight WYSIWYG markdown editor.

It exists as a public delivery channel for Stanza's auto-updater.  The application source is kept private.

## Getting Stanza

Download the latest installer from the [Releases](https://github.com/pelby/stanza/releases/latest) page:

- **macOS Apple Silicon** — `Stanza_*_aarch64.dmg`
- **macOS Intel** — `Stanza_*_x64.dmg`
- **Windows** — `Stanza_*_x64-setup.exe`

Existing Stanza installs update themselves automatically.
